# Linux Configuration Files

This repository contains configuration files and setup scripts for various Linux applications, including **Alacritty**, **Bash**, **Fastfetch**, **Neovim**, **Waybar**, and **clang-format**.

## Usage

   ```bash
   git clone https://github.com/Sora-Fox/configs.git
   cd ./configs && ./setup_all.sh
   ```
